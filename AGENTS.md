# Agent Notes

This repo is intended to be friendly for both humans and coding agents.

## High-signal starting points
- `README.md`
- `LIVE_GENERATION_ARCHITECTURE.md`
- `env-setup-instructions.md`

## Guardrails
- Keep changes minimal and scoped.
- Prefer fixing root causes over patching symptoms.
- Maintain Next.js 15 conventions and existing Tailwind design tokens.

## What to run before opening a PR
- `npm run lint`
- `npm run test:ci`
- `npm run build`
